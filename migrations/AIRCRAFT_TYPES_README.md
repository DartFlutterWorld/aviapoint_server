# Каталог типов самолётов

Этот документ описывает структуру каталога типов самолётов малой авиации.

## Структура базы данных

Таблица `aircraft_types` содержит информацию о типах самолётов с полями:
- `manufacturer` - производитель (Cessna, Piper, etc.)
- `model` - модель (172, SR22, etc.)
- `full_name` - полное название модели
- `category` - категория (single_engine, twin_engine, helicopter, ultralight, etc.)
- `seats` - количество мест
- `engine_type` - тип двигателя (piston, turboprop, jet, turbine)
- `engine_count` - количество двигателей
- И другие поля для технических характеристик

## Источник данных

Данные собраны с сайта [PlaneCheck.com](https://www.planecheck.com/) - международного маркетплейса для продажи самолётов малой авиации.

## Миграции

1. `create_aircraft_types_table.sql` - создание таблицы
2. `insert_aircraft_types_from_planecheck.sql` - заполнение начальными данными

## Использование

После выполнения миграций таблица будет содержать каталог самолётов, который можно использовать:
- В форме создания/редактирования полёта вместо текстового поля
- Для автодополнения при вводе типа самолёта
- Для валидации и фильтрации
